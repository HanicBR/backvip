Algumas das vantagens dos VIPs

  • !noclip — Use o noclip (Voar) mesmo estando no modo PVP
  • !bring <jogador> — Teleporta o jogador até você.
  • !goto <jogador> — Teleporta você até a posição do jogador.
  • !return [jogador] — Retorna o jogador à posição antes de um teleporte.
  • !armor <jogador> <0-255> — Define a armadura do jogador. Útil para desafios e customizações rápidas.
  • !cloak <jogador> [0-255] — Torna o jogador invisível temporariamente.
  • !slap <jogador> [dano] — Dá um tapinha que empurra o alvo; divertido em roleplays leves.
  • @@ <mensagem> (!tsay) — Envia uma mensagem para todo mundo no chat de forma discreta.

Preço: R$10 por 30 dias. Cada R$1 extra adiciona 3 dias de VIP.

O uso excessivo de qualquer um desses comandos para incomodar outros jogadores ou prejudicar o servidor resultará na remoção de todos os seus benefícios VIP.

Como se Tornar VIP

  1. Entre no nosso Discord para iniciar o processo.

    Acessar Discord

  2. Vá para a aba de doações no servidor do Discord.

  3. Preencha o formulário informando para quem é o VIP e os detalhes para a entrega.

  4. Efetue o pagamento via PIX. Você receberá uma chave para a doação. Após o pagamento, envie o comprovante no ticket que foi aberto para você.

  5. Pronto! Seu cargo VIP será ativado em até 24 horas após a confirmação do pagamento.
